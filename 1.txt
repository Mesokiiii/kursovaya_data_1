Тема 6. Турнирная таблица спортивных соревнований
Система ведения турнирной таблицы для футбольной лиги из 20 команд с историей 380 матчей (полный сезон). Автоматический расчет очков по результатам матчей (3 за победу, 1 за ничью, 0 за поражение). Сортировка команд по очкам, при равенстве очков по разнице забитых и пропущенных мячей, затем по количеству забитых мячей. Расчет статистики для каждой команды: количество побед, ничьих, поражений, забитые и пропущенные мячи, разница мячей. Вывод информации о личных встречах между командами. Прогнозирование итоговых мест команд при условии определенных результатов оставшихся матчей. Построение графиков изменения положения команд по турам. Поиск команд по различным критериям.
