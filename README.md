# Football League Tournament Table

Система ведения турнирной таблицы для футбольной лиги из 20 команд с историей 380 матчей (полный сезон).

## Функциональность

- **Турнирная таблица** - Автоматический расчет очков по результатам матчей (3 за победу, 1 за ничью, 0 за поражение)
- **Сортировка** - По очкам, разнице забитых и пропущенных мячей, количеству забитых мячей
- **Статистика команд** - Количество побед, ничьих, поражений, забитые и пропущенные мячи
- **Личные встречи** - Информация о встречах между двумя командами
- **Поиск команд** - Поиск по различным критериям (например, по количеству забитых мячей)
- **Графики** - Построение графиков изменения положения команд по турам
- **История по турам** - Отслеживание изменения позиций команд

## Технологии

- **C# / .NET 8.0**
- **WPF** (Windows Presentation Foundation)
- **OxyPlot** - для построения графиков
- **JSON** - для хранения данных

## Структура проекта

```
project/
├── Models/          # Модели данных (Team, Match, TeamRoundStats)
├── Services/        # Бизнес-логика (LeagueManager, DataLoader)
├── Views/           # UI компоненты (MainWindow, App)
├── Helpers/         # Вспомогательные классы (ChartHelper)
├── Utils/           # Утилиты (Converters)
└── Data/            # Данные (data.json, generate.py)
```

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `FootballLeague.csproj` в Visual Studio
3. Восстановите NuGet пакеты
4. Запустите приложение

## Требования

- .NET 8.0 SDK
- Visual Studio 2022 или выше
